## Možnosti importů

### 1. možnost - delší zápis
``` Python
import data
print(data.mydata)
```

### 2. možnost - doporučovaná
``` Python
from data import my_data
print(my_data)
```

### 3. možnost - moc se nepoužívá
``` Python
from data import *
```

### 4. možnost - alias
``` Python
import data as d
print(d.my_data)
```
